HOW TO BUILD
--------------
Can be built by any Python interpreter by running the main.py with the other source codes (Requires Python)

HOW TO USE
--------------
GUI is very easy to use.
-On the top right, you can add custom points to the points list
-You can select a point from the list on the left, and then remove it using the button on the right named "Remove Selected Point"
-You can generate a set of points whose attributes (number of points, radius) can be decided by you
-You can choose to visualize the triangulations by using the buttons on the bottom, choosing the algorithm you want to use

LIBRARIES USED
--------------
Tkinter
numpy
matplotlib
